Welcome to Apartment Security Mangement System

By using this application it will be easy for the Admin , Guard and Owner to keep track of all the activities performed in apartment virtually. Besides that, they can edit, add and delete the information . The admin is responsible for register Guards and Flat Owner. Guard manages the Visitor in time and out time records, and can also add and view deliveries. This system also will provide a user friendly application system and a security control to ensure the data integrity.
